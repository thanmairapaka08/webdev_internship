# Mini Subscription Management System

A responsive and interactive Subscription Management System built using HTML, CSS, and JavaScript.

This project helps users manage and track their subscriptions in one dashboard with analytics and renewal tracking.

---

## Features

### Authentication
- User Registration
- User Login
- Logout Functionality
- Form Validation

### Dashboard
- Add Subscriptions
- Delete Subscriptions
- Subscription Categories
- Monthly Pricing
- Renewal Date Tracking

### Analytics
- Total Subscriptions
- Monthly Spending Overview
- Active Subscription Count

### UI Features
- Responsive Design
- Professional Light Theme
- Interactive Dashboard
- Clean User Interface

### Storage
- Local Storage Support
- Data persists after refresh

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
