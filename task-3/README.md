# 🎓 Quiz Portal Web App

An interactive quiz application built using HTML, CSS, and JavaScript.  
This project demonstrates core frontend concepts including DOM manipulation, local storage, and responsive UI design.

---

## 🚀 Features

- 🔐 User Login & Logout
- 📚 Category Selection (HTML, CSS, JavaScript)
- 🎯 Random 5 Questions per Quiz
- 🔀 Shuffled Questions & Options
- ⏱ Timer-Based Quiz
- 📊 Score Calculation
- 🏆 Leaderboard using localStorage
- 🔁 Play Again Functionality
- 🎨 Modern UI Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
