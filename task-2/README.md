# 🌐 Task-2: Responsive Registration Form & To-Do List

## 📌 Project Overview
This project is part of my Web Development Internship (Task-2).  
It demonstrates intermediate skills in HTML, CSS, and JavaScript including form handling, validation, and DOM manipulation.

---

## 🚀 Features

### 🔹 Registration Form
- Modern UI design
- Single-column layout
- Real-time validation (while typing)
- Input error handling
- Password & confirm password validation

### 🔹 To-Do List
- Add tasks dynamically
- Delete tasks
- Interactive UI

### 🔹 Responsive Design
- Works on mobile, tablet, and desktop
- Built using:
  - Flexbox (alignment)
  - CSS Grid (layout)
  - Media Queries

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
