# Task-4 Full Project Implementation

This project is developed as part of the Web Development Internship Task-4.

## Project Overview

The project includes:

- Personal Portfolio Website
- Product Listing Page
- To-Do List Application
- Responsive Design
- JavaScript Functionality
- Local Storage Support

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

### Portfolio Website
- Professional responsive design
- About section
- Projects section
- Contact form with validation
- Interactive buttons and navigation

### Product Listing Page
- Dynamic product display
- Category filtering
- Price sorting
- Responsive product cards

### To-Do Application
- Add tasks
- Delete tasks
- Data stored using Local Storage
- Tasks remain after page refresh
